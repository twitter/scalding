ReplImplicits

object ReplImplicits extends FieldConversions

Object containing various implicit conversions required to create Scalding flows in the REPL. Most of these conversions come from the Job class.

    Rather than give the full power of cascading's selectors, we have a simpler set of rules encoded below: 1) if the input is non-definite (ALL, GROUP, ARGS, etc...) ALL is the output.
