ArrayWrappingOutputStream

Related Doc: package JavaStreamEnrichments

class ArrayWrappingOutputStream extends OutputStream

Wraps an Array so that you can write into it as a stream without reallocations or copying at the end. Useful if you know an upper bound on the number of bytes you will write
